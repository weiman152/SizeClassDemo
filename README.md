# SizeClassDemo
SizeClass是一种很好用的布局方式，方便在不同的手机上显示不同的效果，甚至显示不同的控件。

SizeClass是在storyboard设置不同显示样式的一种方式。用得恰当，可以解决很多问题，省去写代码的麻烦，提高开发效率。

环境：
xcode9 ，ios8以上
